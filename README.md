## Installation Instructions

To install Code Monsters on your Android device, you'll need to allow installation from unknown sources:

### For Android 8.0 and higher:
1. Download the APK file from the latest release
2. Open the APK file on your device
3. When prompted, tap "Settings" to allow installation from this source
4. Toggle on "Allow from this source"
5. Go back and tap "Install"

### For Android 7.1 and lower:
1. Go to **Settings** > **Security**
2. Enable **Unknown Sources** (allow installation of apps from sources other than the Play Store)
3. Download the APK file from the latest release
4. Open the APK file and tap "Install"
